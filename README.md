
Build Setup

# install dependencies
cnpm i
(可以用cnpm或yarn，更方便快捷，你值得拥有)

# serve with hot reload at localhost:8564
npm run dev

# build for production with minification
npm run build


技术栈

vue(数据绑定)

vue-router

vuex(管理组件状态，实现组件通信）

webpack(打包工具)

stylus（CSS 预处理语言）

axios（是一个基于Promise 用于浏览器和 nodejs 的 HTTP 客户端，做数据交互）
